알림이 오지 않아요▼
크롬 익스텐션이 설치되어 있는지 확인해주세요. 설치 후 구글 계정으로 로그인하면 자동으로 알림이 시작돼요.
댓글에 말풍선이 안 생겨요▼
구글 로그인 후 로그아웃 후 다시 로그인 해보세요. YouTube Data API 권한이 필요해요.
구글 로그인이 안 돼요▼
익스텐션을 삭제 후 다시 설치해보세요. Cloud Console에서 본인 이메일이 테스트 사용자로 등록되어 있는지 확인해주세요.
이메일 알림이 안 와요▼
Gmail API 권한이 필요해요. 로그아웃 후 다시 로그인하면 새 권한이 적용돼요.
Pro 구독은 어떻게 하나요?▼
현재 Pro 구독 기능을 준비 중이에요. 월 1,490원 / 연 14,900원으로 제공될 예정이에요.
크롬 익스텐션 없이도 사용할 수 있나요?▼
아니요, BubbleTag는 크롬 익스텐션이 반드시 필요해요.
방해금지 중에도 중요 알림이 와요▼
방해금지 설정에서 "중요 알림 허용" 옵션을 OFF로 바꾸면 중요 알림도 차단할 수 있어요.
